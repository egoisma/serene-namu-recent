# serene-namu-recent

나무위키 최근편집 항목들을 더 역동적으로 보여줍니다.

## External Python modules

* flask
* flask_restful
* requests
* beautifulsoup4
* datetime
* pytz

## Executing API Server

You have to install external Python modules first.

```
python server/server.py
```
