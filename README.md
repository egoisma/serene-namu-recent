# serene-namu-recent

나무위키 최근편집 항목들을 더 역동적으로 보여주는 웹어플리케이션입니다.

## Required Python modules

* flask
* flask_restful
* requests
* beautifulsoup4
* datetime
* pytz

## Executing API Server

You have to install required Python modules first.

- <code>python server/server.py</code>

## Front-end

* frontend/index.html
